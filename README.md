## Арифметический тренажёр (Django)

Учебный проект по курсу Python: **веб-приложение на Django** на образовательную тематику.

### Цель проекта

Сделать локальный веб-сервис-тренажёр для практики **сложения, вычитания и умножения** с постепенным повышением сложности и сохранением прогресса.

### Функциональность

- **Главная страница**: описание и текущая статистика.
- **Тренировка**: примеры по уровню, проверка ответа, повышение уровня каждые 5 верных подряд.
- **История**: последние попытки (пример → ответ → верно/неверно).
- **Статистика**: сводка и разбивка по операциям.
- **Настройки**: никнейм и выбор операций (нельзя отключить все типы примеров).

### Хранение данных приложения (без «своей» SQL-логики)

Прогресс тренажёра хранится в **JSON-файлах** в каталоге `data/users/`:

- файл на пользователя: `data/users/<uuid>.json`;
- идентификация через cookie `trainer_uid`.

При первом запуске Django может создать локальный **`db.sqlite3`** для стандартных компонентов (сессии и т.д.) — это служебный файл, не логика тренажёра.

### Технические требования (критерии задания)

- HTML-шаблоны с наследованием: `templates/base.html` и страницы в `templates/trainer/`.
- Стили: `static/css/styles.css`.
- Не менее 4 страниц, 2+ форм с валидацией, 2+ страниц вывода информации.

### Установка и запуск (Windows)

1) Установите зависимости:

```bash
python -m pip install -r requirements.txt
```

2) Примените миграции (для встроенных приложений Django):

```bash
python manage.py migrate
```

3) Запустите сервер:

```bash
python manage.py runserver
```

4) Откройте в браузере `http://127.0.0.1:8000/`.

### Проверка pylint (опционально)

```bash
pylint config trainer
```

Конфигурация: файл `.pylintrc` в корне проекта.
